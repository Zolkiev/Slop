# Slop Games

Collection de petits jeux web/mobile simples mais addictifs ("slop games").

## Structure

```
Slop/
├── _starter-template/   # Squelette réutilisable (à venir)
├── 1st_Slop/            # Jeu 1
└── ...                  # Un dossier par jeu
```

Chaque jeu est autonome : son propre code, ses assets (générés via Pixellab) et son déploiement.
